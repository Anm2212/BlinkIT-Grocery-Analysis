# 🛒 BlinkIT Grocery Sales Analysis

> An End-to-End Data Analytics & Machine Learning Project using **Excel, PostgreSQL, Power BI, and Python**.

![Power BI Dashboard](Images/PowerBI%20dashboard.png)

---

# 📌 Project Overview

This project presents a complete data analytics pipeline using the BlinkIT Grocery Sales dataset. The workflow includes data cleaning, business analysis with SQL, dashboard development in Power BI, Excel-based reporting, and Machine Learning for predictive analysis.

The objective is to extract meaningful business insights, evaluate outlet performance, analyze customer purchasing patterns, and demonstrate an end-to-end analytics workflow using industry-standard tools.

# 📈 Key Business Insights

- Supermarket Type1 generated the highest revenue.
- Tier 3 outlets contributed the highest sales.
- Medium-sized outlets achieved the best overall performance.
- Fruits & Vegetables and Snack Foods were among the top-selling categories.
- Average customer ratings remained consistent across outlet types.
- Sales trends indicate steady growth across newly established outlets.
